# v3 sound candidates — audition pool

Each subdirectory holds 5 candidates per event. For `flap` and `coin`
each candidate is a 3-variant pack (`_a/_b/_c`) — pick the *family
number* and you get the whole pack.

Listen with `ffplay`, VLC, or any player. Reply with picks like
`flap=2, coin=3, mushroom=1, ...`.
